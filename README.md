# Craps Simulator
An all encompassing simulator with GUI and statistical analysis to compare craps strategies in the long run.

Features:
1) Ability to play craps visually
- Simulate a craps board with a GUI
- Simulate Game Logic with correct pay tables
- Be able to adjust table rules like minimums, maximums, and payouts on the field

2) Track Bankroll Information
- Track each roll, how much was won or lost
- Track for a shooter how much was won or lost
- Visualize bankroll over time on a graph

3) Strategy Teseting
- Be able to define a strategy, and automatically run it for a set number of rolls
- Expected value calculations for a strategy
- Graph strategies against one another for the same roll period

4) Desktop Application and tech stack
- This should be a desktop application
- It should be compatible with Windows and Linux
- It can likely use an SQlite DB for storing the data
- It should use rust or python
